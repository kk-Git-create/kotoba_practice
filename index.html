フォルダ構成
このアプリは index.html・words.json・画像ファイルを同じフォルダに入れて使います。
更新するときは words.json と画像ファイルを差し替えるだけです。
📁 kotoba_practice/
　├── index.html　　← このファイル
　├── words.json　　← 問題データ
　├── 01_zaru.png
　├── 02_shamoji.png
　└── ...
words.json の形式
[ { "id": 1, "word": "ざる", "reading": "ざる", "image": "01_zaru.png" }, { "id": 13, "word": "スタンプ", "reading": "すたんぷ", "image": "13_stamp.png", "aliases": ["はんこ", "印鑑", "いんかん"] } ]
・word：漢字またはひらがな（正解判定に使用）
・reading：必ずひらがな（揺れ吸収のため）
・image：同じフォルダにある画像ファイル名
・aliases：別名・同義語リスト（省略可）。例：スタンプ→はんこ
Geminiへのプロンプト（コピペ用）
① JSONを作成する
「失語症の呼称訓練用に、日常的だが使用頻度がやや低い物品の名称を20個選び、以下のJSON形式で出力して。
[{"id":1,"word":"ざる","reading":"ざる","image":"01_zaru.png"}, ...]
imageはid2桁_ローマ字.png形式。」

② 画像を生成する（1枚ずつ）
「白背景で、『〇〇』を中央に大きく描いたシンプルなイラスト。リハビリ用絵カード。文字・影・装飾は不要。」
